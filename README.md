# HE/CLA Frontend #

*See repository [hecla-runtime-environment](https://www.github.com/sjkorpela/hecla-runtime-environment) for full project description!*

### How to run locally ###

Needs:
* NPM to build and run

Instructions:
1. Run command `npm install` to install dependencies.
2. Run command `npm run dev` to run.
